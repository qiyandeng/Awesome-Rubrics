# Awesome-Rubrics

A Docusaurus-powered GitHub Pages site for collecting rubric-related papers, surveys, datasets, benchmarks, and tools.

## Local development

```bash
npm install
npm run start
```

## Production build

```bash
npm run build
```

## GitHub Pages

This repository is configured for GitHub Pages deployment with:

- `url`: `https://qiyandeng.github.io`
- `baseUrl`: `/Awesome-Rubrics/`
- GitHub Actions workflow: `.github/workflows/deploy.yml`

After pushing to `main`, enable GitHub Pages in the repository settings and choose **GitHub Actions** as the source.
