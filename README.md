# Awesome-Rubrics

A curated academic hub for rubric-related research, organized by generation strategy, model training stage, and use case.

**Live site → [https://qiyandeng.github.io/Awesome-Rubrics/docs/awesome-list](https://qiyandeng.github.io/Awesome-Rubrics/docs/awesome-list)**

## What's inside

- **Overview table** — all papers at a glance, with categorization tags (how rubrics are used, model training stage) and direct source links
- **Annotated reading list** — taxonomy-grouped cards with core idea, why it matters, and expandable annotations
- **Survey overview** — organizing axes across use cases, model stages, and generation strategies


## Contributing

To add a paper, edit [`docs/awesome-list.mdx`](docs/awesome-list.mdx):

1. Add an entry to the `papers` array at the top for the overview table
2. Optionally add a `<PaperCard>` inside the matching `<PaperGroup>` for a detailed annotation
3. Open a pull request

## Deployment

Pushes to `main` automatically trigger GitHub Actions (`.github/workflows/deploy.yml`) to build and deploy to GitHub Pages.
